#overview
消息总线管理系统用于完成对消息总线的管理、监控、参数配置等相关功能

##技术说明

* 数据存储：mysql
* Ioc: Spring
* 表现层框架：Struts 2
* 持久层框架：mybatis
* 前端组件：jTable,jQuery

##功能模块

* 信息维护
* 监控管理
* 预警管理
* 配置管理
* 权限管理
